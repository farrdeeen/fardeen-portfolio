## ✨ Features

- 🖥️ Responsive landing page with background image and profile picture
- 📩 Contact form connected to FastAPI backend
- 💾 Messages saved to SQLite database
- 🔐 Admin page to view submitted messages
- 🔄 Full API tested via Swagger Docs
- 🎯 Deployed using modern tooling (coming soon)
